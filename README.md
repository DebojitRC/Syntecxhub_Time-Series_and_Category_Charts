# Syntecxhub Time-Series & Category Charts Analysis

Data Science Week 2 Task 1

Author: Debojit Roy Chowdhury

------

## Overview
This project focuses on analyzing sales data using time series and category-based visualizations. It demonstrates how data can be transformed into meaningful insights through graphical representations.

The analysis includes line charts for trends over time, bar charts for category comparison, and pie charts for distribution analysis.

## Objectives
* Perform time series analysis using sales data
* Aggregate data on a monthly basis
* Compare sales across different product categories
* Visualize category distribution using pie charts
* Save charts as image files for reporting
* Interpret and communicate insights from visualizations

## Dataset
The dataset used is a retail sales dataset containing transactional information such as:
* Order Date
* Category
* Sales
* Customer details and product information
This dataset is publicly available and widely used for sales analysis tasks.

## Technologies Used
* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Key Analysis

### Time Series Analysis
* Converted `Order Date` to datetime format
* Aggregated sales data monthly
* Plotted a line chart to visualize sales trends over time

### Category Comparison
* Grouped data by product category
* Used a bar chart to compare total sales across categories

### Distribution Analysis
* Visualized category-wise contribution using a pie chart
* Displayed percentage share of each category

## Visualizations

### Monthly Sales Trend
* Shows how sales change over time.

### Sales by Category
* Compares total sales across categories.

### Category Sales Distribution
* Displays the proportion of sales contributed by each category.

## Key Insights
* Sales show noticeable fluctuations over time, indicating seasonal trends
* The Technology category contributes the highest share of total sales
* Furniture and Office Supplies also contribute significantly
* There is a clear variation in performance across categories
